**[[Month 1]]** - **MPLS Fundamentals**
Build core MPLS knowledge using Juniper-specific protocols and configurations.
Review MPLS basics: RSVP-TE for label distribution (preferred over LDP in Juniper), signaling, and forwarding (ingress/egress/transit). Study BGP essentials including groups, policies, and next-hop self.
Allocate 10-15 hours weekly to Junos MPLS User Guide sections on enabling MPLS interfaces and basic RSVP sessions via set protocols mpls interface all.
 
**[[Month 2]] - MP-BGP and VPNs**
Configure MP-BGP for MPLS VPN services on Juniper PE routers.
Set up Layer 3 VPNs with route targets, route distinguishers, and family inet-vpn unicast; integrate with CE protocols like OSPF using sham links. Create a lab topology with vMX routers simulating a 4-node MPLS core.
Spend 20 hours weekly practicing set protocols bgp group vpn type internal family inet-vpn unicast and VRF instance policies.
 
**[[Month 3]] - Advanced Features and Troubleshooting**
Tackle traffic engineering, Layer 2 VPNs, and real-world optimization.
Implement Fast Reroute (FRR), class-of-service (CoS) over MPLS, and EVPN for L2 services; troubleshoot with show route label-switched-path, monitor traffic, and BGP session diagnostics. Simulate ISP scenarios with link failures.
Finish with self-assessments via end-to-end traceroutes and verification of MP-BGP NLRI exchanges.
